Front-End

O front-end está muito relacionado com a interface gráfica do projeto em que o usuário irá interagir diretamente.

Num ambiente para internet, compreende os profissionais que desenvolvem os códigos para criar a parte visual das páginas.

Isso significa que o front-end se relaciona com as partes dos sites que mais se aproximam da pessoa usuária.

Quem trabalha com front-end desenvolve a interface gráfica através de tecnologias bases da web: HTML, CSS e JavaScript.
O desenvolvimento web front-end envolve o lado do cliente(client-side) e é executado por um navegador de internet (browser).

Origem

Com o passar do tempo, os sistemas ficaram tão complexos, que é necessário que pessoas com especialidades diferentes se responsabilizem por partes específicas do desenvolvimento de sites. Antigamente, utilizava-se o termo webmaster para definir um profissional capaz de gerenciar as tarefas tanto de um desenvolvedor quanto de um designer.

Hoje em dia, as atividades focam em experiência de usuários e compreendem usar alguma ferramenta visual, como Photoshop, Illustrator, Adobe XD, Figma, Sketch etc. São profissionais especializados em conceitos como paleta de cores, desenhos, fontes e tipografias para desenhar as telas. Já os profissionais de front-end desenvolvem os códigos para criar telas.

Dicas para um desenvolvedor Front-End:

  1. Estudar e praticar os princípios básicos: HTML e CSS
  2. Conhecer Estrutura de Dados e Algoritmos com JavaScript
  3. Criar sites responsivos, adotando o "Mobile First"
  4. Priorizar a acessibilidade nas páginas
  5. Realizar testes unitários para verificar o comportamento das menores partes do código

Linguagens Front-end:

Principais atividades:

  1. Estruturar layouts
  2. Organizar Conteúdos
  3. Posicionar menus e botões
  4. Definir cores e fontes
  5. Criar animações
  6. Implementar recursos interativos
  7. Aplicar técnicas de otimização de desempenho

Diferenças

Front-end
Cuida de toda interface de uma aplicação que possui interação com o usuário.
Back-end
Cuida da parte do sistema que não é visível para os usuários finais: regras de negócio, processamento de dados, segurança, interação com bancos de dados e servidores.
Full Stack
Profissional que trabalha tanto com front-end, quanto com bach-end.
